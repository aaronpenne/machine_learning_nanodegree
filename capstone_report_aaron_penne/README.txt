This project is a classifier for applications to DonorsChoose.org. It uses NLP and other techniques to solve the classification problem. The data and Kaggle competition are here: https://www.kaggle.com/c/donorschoose-application-screening

To run this project simply run donorschoose.py with Python 3+

Libraries used are:
- os
- datetime
- numpy (https://github.com/numpy/numpy)
- pandas (https://github.com/pandas-dev/pandas)
- wordcloud (https://github.com/amueller/word_cloud)
- matplotlib (https://github.com/matplotlib/matplotlib)
- seaborn (https://github.com/mwaskom/seaborn)
- nltk (https://github.com/nltk/nltk)
- TextBlob (https://github.com/sloria/TextBlob)
- sklearn (https://github.com/scikit-learn/scikit-learn)

Data is expected to be in the 'data' directory and contain the following files:
- resources.csv
- train.csv
- test.csv
